== Description ==

This plugin allows you to integrate with Cloudanix. 

Going live with your website is not enough. You need good operations support too. We at Cloudanix have built 7 diverse set of tools - which you need to protect your brand and reputation and provide your customers an experience they expect. These tools also ensure that you stay ahead in the race with your competition.

We have built the following "recipes", as we call it:
Uptime Monitoring, Speed Monitoring, DNS Monitoring, Privacy Score, Blacklist Monitoring, Site Health and OWASP Top 10

This plugin for Wordpress let's you connect and configure different recipes with [Cloudanix](https://www.cloudanix.com) for your site via API. You can view all the results at [Cloudanix](https://www.cloudanix.com). Wordpress dashboard would have a widget to connect you with the Cloudanix dashboard.

== Installation ==
Automatic installation

1. Log into your WordPress admin
2. Click __Plugins__
3. Click __Add New__
4. Search for __Cloudanix__
5. Click __Install Now__ under "Cloudanix (Website Operation Tools)"
6. Activate the plugin

Manual installation:

1. Download the plugin
2. Extract the contents of the zip file
3. Upload the contents of the zip file to the wp-content/plugins/ folder of your WordPress installation
4. Then activate the Plugin from Plugins page.


== Frequently Asked Questions ==

= I installed Cloudanix, what now? =

Open the **Tools** menu, then open **Cloudanix** then find the Cloudanix Dashboard details.

= How can i view this in Wordpress Dashboard? =

On the Wordpress Dashboard, you should find a **Cloudanix** widget. Move that to the top to view Cloudanix Dashboard details.
